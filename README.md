# Productivity Tracker

Allows to track user keyboard, mouse and touchpad activity

## Prerequisites

**Currently only Linux is supported.**

See `.env.example` for required environment variables.

```bash

- xinput v1.6.3
- awk v1.3.4
- sed v4.8
- node.js v18.17.1
- Google Sheets API v4 credentials
```

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```
